Command action data Description
QUERY JSOUP query Execute JSOUP query.
TEXT none Select content as string.
HTML none Select content as html string.
ATTRIBUTE name of the attribute/td> Select attribute of given name.
OUTPUT predicate Create statement with input value as an object, as a subject use given predicate and as a subject used last defined subject.
SUBJECT [optional]predicate Create new subject used by {@link #OUTPUT}. If action data are set then, connect the new subject to the current subject with given action data (predicate). If action data are left empty, then connect the new subject to the current parent subject.
UNLIST none Given list of elements put each element into separated group.
SUBJECT_CLASS class uri Set given URI as a subject class.
If 'Advanced settings'.'Default has predicate' is set then given value is used action data for all SUBJECT actions.;
