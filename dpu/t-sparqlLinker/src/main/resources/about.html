This DPU is designed to execute construct asymetrically on inputs.
List of inputs:
  • perGraph-Input - data to link.
  • reference-Input - data to link to.
Common usage is to link data (perGraph-Input) to reference data (reference-Input).
If per-graph options is not checked then this DPU act as t-sparqlConstruct with per-graph unchecked.
If per-graph is not checked then following work flow is used:
For each input graph (perGraph-Input) G and all reference graphs (reference-Input) R given SPARQL is executed over G unified with R.
