This DPU can be used to link data (perGraph-Input) to reference data (reference-Input), by asymmetric execution of given SPARQL construct.

List of inputs:
  • perGraph-Input - data to link.
  • reference-Input - data to link to.
The way in which the given SPARQL Construct is executed depends on per-graph option.
  • not checked - this DPU act as t-sparqlConstruct with per-graph unchecked.
  • is checked then following work flow is used:
    For each input graph (perGraph-Input) G and all reference graphs (reference-Input) R given SPARQL is executed over G unified with R.
